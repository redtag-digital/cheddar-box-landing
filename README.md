# Cheddar Box Cafe Landing Page

A high-quality, responsive landing page for The Cheddar Box Cafe in Louisville, KY. Built with a focus on speed, appetizing design, and lead generation.

## Features

- **Responsive Design:** Built with Tailwind CSS for a seamless experience across all devices.
- **Lead Generation:** Integrated Netlify form for the "Free Lunch for a Month" promotion.
- **Custom Tracking:** GA4 event tracking for form submissions and menu interactions.
- **Authentic Content:** Features real menu staples like Honey Pecan Chicken Salad and Tomato Artichoke Soup.
- **Official Rules:** Dedicated rules page for the promotion.

## Tech Stack

- **HTML5/CSS3**
- **Tailwind CSS** (via CDN for rapid deployment)
- **Netlify Forms**
- **Google Analytics 4**

## Setup & Deployment

1. **Local Development:** Simply open `index.html` in any browser.
2. **Deployment:** Designed for effortless deployment on **Netlify**. 
   - Ensure "Form detection" is enabled in your Netlify build settings to capture drawing entries.
   - Update the GA4 placeholder ID (`G-XXXXXXXXXX`) in `index.html` with your live measurement ID.
